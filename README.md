> The core logic for Typings.

## Usage

```ts
import * as typings from 'typings-core'
```

## Contributing

```sh
# Installation
# Fork this repo (https://github.com/typings/typings-core)
# Clone the fork (E.g. `https://github.com/<your_username>/typings-core.git`)
cd typings-core
npm run bootstrap

# Build
npm run build

# Test
npm run test
```
